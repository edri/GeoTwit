Help and FAQ

This page contains all you need to know about GeoTwit!

Okay, what is GeoTwit?

GeoTwit is an application, which allows you to analyze activities on Twitter-based subjects with maps and charts, either on real-time or staticly.
You can indeed find two modes of search: a dynamic mode (for real-time activities) and a static mode.

How is the dynamic mode working?

In order to follow a streaming process, you have to indicate at least one keywords set and to choose an area on the map (either with the drop-down menu or by manually drawing a rectangle on it). You can also select the language in which you want the received Tweets to be written in.
If you fill a second keywords set, a second streaming process will simultaneously be started so you can compare the two subjects.

Once you started the streaming(s), you will be able to access both the map in which the received Tweets are displayed and some charts that allow you to better analyze and understand the incoming data.

In addition of these features, you can also import the results: once you stopped the streaming process, you will automatically be asked to either export or not the data. If you choose "yes", you will be able to download a text file containing the data.

How is the static mode working?

Like in the dynamic mode, you have to indicate at least one keywords set and to choose an area on the map (by manually drawing a circle on it only). In addition, you also have to select a range of dates in which the received Tweets will be contained. You can also select the language in which you want the received Tweets to be written in.
If you fill a second keywords set, a second static search will simultaneously be performed so you can compare the two subjects.

Once you searched the results, you will be able to access the map in which the received Tweets are displayed.

How to use the application?

Firstly, you have to possess a Twitter account.
Then, click on the "Get Started" button or on the "Connect" link of the home page, allow GeoTwit to access your account, and that's it! You can now use GeoTwit.

Can you access my Twitter's password?

Nope, we only get your username, in order to display it in the header of the pages.
You account is used to make the different search requests of the application.

Is the application free?

Yes, GeoTwit is entirely free.
